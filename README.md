# Appscrip Frontend Assignment

A Product Listing Page built using **HTML, CSS, and JavaScript**. It fetches products from FakeStore API and displays them in a responsive grid.

## Features
- Fetch products from FakeStore API  
- Search and category filter  
- Responsive grid layout  
- Dynamic product rendering with JavaScript  

## How to Run
Open `index.html` in a browser or run `python -m http.server 8000`.
